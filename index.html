GitHub 简介与资源！

什么是 GitHub？

GitHub 是一个使用 Git 进行版本控制的协作平台。GitHub 是共享和贡献开源软件的热门场所。

什么是仓库？

仓库是包含文件和文件夹的项目。存储库跟踪文件和文件夹的版本。有关更多信息，请参阅 GitHub Docs 中的“ 关于仓库 ”。

什么是分支？

分支是存储库的并行版本。默认情况下，您的仓库有一个名为 main 的分支，它被视为最终分支。

通过创建其他分支，您可以复制存储库的主分支并安全地进行任何更改，而不会中断主项目。

许多人使用分支来处理特定功能，而不会影响项目的任何其他部分。有关详细信息，请参阅 “ 关于分支 ”。

什么是配置文件 README？

配置文件 README 本质上是 GitHub 个人资料上的“关于我”部分，您可以在其中与 GitHub.com 上的社区分享有关您自己的信息。

GitHub 在个人资料页面顶部显示您的个人资料 README。有关详细信息，请参阅 “ 管理配置文件 README”。

⌨️ Activity: Your first branch

打开一个新的浏览器选项卡，并导航到您新创建的存储库（本练习的副本）。然后，在阅读此选项卡中的说明时，执行第二个选项卡中的步骤。

  1. 导航到存储库标题菜单中的 <> 代码 选项卡。
  2. 单击 main 分支下拉列表。
  3. 在文本框中输入 my-first-branch。
  4. 单击 Create branch： my-first-branch from main 以创建您的分支。
