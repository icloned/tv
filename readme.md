# icloned-tv

一个纯静态页面仓库，当前以单页形式提供内容展示，仓库根目录直接作为站点发布目录使用。

## 项目说明

- 站点入口文件为 `index.html`。
- 当前仓库未引入构建工具、包管理器或前端框架，修改后可直接提交发布。
- 页面内容主要用于内部路由场景，公开访问能力和可用性不做保证。
- 仓库中涉及的外部资源链接与展示内容需要由维护者自行确认合规性与可访问性。

## 目录结构

```text
.
├─ index.html    # 站点主页面
├─ CNAME         # 自定义域名配置
├─ robots.txt    # 爬虫访问规则
└─ readme.md     # 项目说明文档
```

## 部署方式

当前仓库适合直接通过静态托管方式部署，例如：

- GitHub Pages
- Nginx 静态目录
- 内网 Web 服务或对象存储静态托管

如使用 GitHub Pages，自定义域名配置位于 `CNAME`，当前值为 `iclone.dpdns.org`。

## 维护建议

1. 修改页面内容时，优先直接编辑 `index.html`。
2. 如果新增静态资源，建议按 `assets/`、`images/` 等目录分类，避免后续维护混乱。
3. 若页面用途发生变化，应同步更新本说明文档，保持部署方式与访问范围描述一致。

## 备注

仓库现有内容偏向内部使用。若后续要对公网开放，建议补充：

- 明确的版权与内容来源说明
- 更完整的项目用途介绍
- 基本的安全与可维护性整理
